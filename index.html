About

Agilita Solutions provides Audio Visual Systems Programming with a focus on practical, maintainable deployments—especially Crestron-based systems.

Whether you need a clean template approach across multiple rooms, help commissioning, or improvements to stability and UX, I can support end-to-end delivery.

Typical work

  • Crestron programming (SIMPL / SIMPL+ / SIMPL#)
  • Device integrations (HTTP, TCP/UDP, RS-232, APIs)
  • Commissioning, snagging & handover packs
  • Standardised UI flows and room templates
